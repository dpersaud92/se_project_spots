# **Spots**

A dynamic web-based photo-sharing platform.

## **Description**

Spots is an intuitive, web-based photo-sharing application where users can upload photos and share them with others. Visitors to a userâ€™s profile can view and like their photos, providing a simple yet interactive social experience.
